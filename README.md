# Find User

Проект содержащй информацию о многих странах!

## Технологии

- React
- TypeScript
- SCSS
- API-REST

### Возможности

- Смотреть информацию о пользователях GitHub

- Поиск пользователя

- Переключение тем : светлой/темной
